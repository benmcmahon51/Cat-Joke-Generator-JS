# JS-Cat-Joke-Generator
## Description
Cat Joke Generator is an independent CodeCademy project with the goal of creating a JavaScript file that generates a random response. 
## Features
* Cat Joke Generator contains an array of 10 cat jokes.
* A switch pairs each array item with a punchline.
* A function is utilized to ensure that the same joke is not delivered twice in a row.
## How to use
1. Clone the code repository of index.js 
2. Install dependencies.
3. Run the JavaScript in terminal, each time the code runs, a random cat joke will be returned!
